# Introduction to Langchain, OpenAI and Ollama

To run streamlit in local machine:

Step 1:

```bash
conda create -p venv python==3.10 -y
```
Step 2:
```bash
conda activate venv\
```
Step 3:
```bash
pip install -r requirements.txt
```
Step 4:
```bash
cd Langchain\ollama
```
Step 5:
```bash
streamlit run app.py
```
